# 锚点笔记 (Anchor Notes)

## 软件用途
锚点笔记是一款基于 Tauri + Rust 构建的纯本地桌面笔记软件。它的核心机制是将所有笔记及其相关的图片、音视频附件完全打包存储在本地硬盘上，不进行任何云端同步，以确保数据的绝对私密和安全。

## 主要功能
- **富文本编辑**：基于定制的 Quill.js，支持基础的文字排版。
- **丰富的媒体支持**：
  - **图片**：支持拖拽或粘贴插入，可右键切换多种尺寸比例（1x 到 16x）。
  - **音频**：支持插入本地音频，自动解析音频内置封面。支持胶囊到方形小卡等多种展示形态，自带波形进度条并支持点击波形精准跳转播放。
  - **视频**：支持插入本地视频文件，提供悬浮式的独立视频播放器。
  - **内置录音**：可在编辑器内直接调用麦克风进行语音录制，录完直接生成可播放的音频块。
- **本地化存储与加密**：每篇笔记都有自己独立的文件夹来存放 SQLite 数据库文件（文本）和相关附件目录。内置 AES-GCM 加密，可加密存储。

- **历史回溯**：独立保存操作增量（Delta），可实现笔记历史时间线的回溯。
- **导出与备份**：可一键将整篇笔记（包含所有文字和多媒体附件）打包导出为 `.tar.gz` 压缩包。

## 如何使用（开发与编译）

本项目为 Rust Cargo 工作空间，主要 UI 代码在 `anchor_ui` 目录下。

**环境要求**
- 安装 Rust (cargo)
- 安装 Node.js
- Windows C++ 生成工具 (Tauri 编译需要)

**调试运行**
```bash
cd anchor_ui
cargo run
```

**编译打包为 EXE**
```bash
cd anchor_ui
cargo build --release
```
编译完成后，生成的可执行文件位于 `target/release/anchor_ui.exe`。
