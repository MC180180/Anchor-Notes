# 项目技术栈 (Technology Stack)

本项目使用以下核心技术进行构建：

## 1. 渲染引擎 (Rendering Engine)
- **winit**: 用于跨平台窗口创建、管理以及事件处理（键盘、鼠标等输入）。
- **wgpu**: 基于 WebGPU 标准的现代、跨平台图形和计算 API，作为项目的核心渲染引擎，提供高性能的底层硬件加速图形渲染。

## 2. 数据存储与交互引擎 (Data Storage & Interaction Engine)
- **Tauri**: 用于构建更小、更快、更安全的桌面应用程序的框架。负责管理前端用户界面，并提供前端与 Rust 核心逻辑之间的高效通信（IPC）和交互。
- **SQLite**: 轻量级、嵌入式的关系型数据库。与应用系统深度集成，负责核心业务数据的本地持久化存储、查询与管理。
